Visual Studio - Turning nothing into something

Computer science and the programming courses have taught me that there is a simple solution to most problems we face on a day to day basis. Creating UI's(User Interfaces) which interact with the user to collect information and promp a solution or answer can help ones life tremendously. Im currently working on a program that takes a customers information like first name, last name, and address and creates a list of customers that have either a checking or savings account and allows the user to view the customers information and allows them to either withdraw or deposit into their account.
